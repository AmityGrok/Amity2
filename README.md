## Adding Content

### Adding Conversations

1. Add entry to `_data/conversations.json`
2. Create markdown file in `_content/conversations/`
3. Site auto-indexes on refresh

### Adding Screenshots

1. Place images in `assets/media/screenshots/[YYYY-MM]/`
2. Add metadata entry to `_data/media.json`
3. Gallery auto-updates

### Adding Timeline Events

1. Edit `_data/timeline.json`
2. Include date, event, significance, and source
